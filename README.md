# SAR Transaction Analysis

SQL-style analysis of SAR transaction data using Pandas.

### What it does
- Loads SAR data from CSV
- Groups by velocity_flag to count alerts
- Filters for flagged transactions
- Finds largest flagged transactions by amount

### Result
Found 0 flagged transactions out of 7 in sample dataset.
